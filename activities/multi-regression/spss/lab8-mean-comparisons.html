Step 3 - Use a paired samples t-test to compare SUVR in the "Deprived" condition to SUVR in the "Rested" condition

  1. Go to Analyze->Compare Means->Paired Samples T Test
  2. Move SUVR_Deprived to "Variable 1" for Pair 1
  3. Move SUVR_Rested to "Variable 2" for Pair 1
  4. Check the box for effect size, and select "Standard deviation of the difference" (this will give you effect size estimates specific to the within-subject design)
Now look at the output
  • Focus on the "Paired Samples Statistics", the "Paired Samples Test", and the "Paired Samples Effect Sizes" tables (don't get confused by the "Paired Samples Correlation" table, this descibes the correlation between the two measures, not the difference)
  • The "Paired Samples Statistics" table includes the Means, N (number of observations), and standard deviations for each condition, you will need this info to complete Response 4 below"
  • The "Paired Samples Test" table includes the Mean of the difference between pairs, 95% confidence interval around the mean, t-statistic, and significance (p-value)
    • The t-statistic is the difference between means divided by the standard error of the difference (we will go over this idea in more depth next week)
    • The t-statistic is used to compute a significance value (p-value) that is **the probability of observing a difference at least far from zero (as the difference observed in this sample) if the null hypothesis is true (i.e., there is no true population difference in SUVR after sleep deprivation vs rest) - if the two-sided p-value is low (usually the threshold is .05), then we say there is a "significant difference" between the conditions. For this activity, ignore the "One-sided p" and only pay attention to the "Two-sided p".
  • With this design, observations are not independent. Each observation is related to another observation from the same individual.

  • A paired samples t-test is exactly the same as conducting a 1 sample t-test on the difference between conditions compared to zero (notice that the output says "mean of the differences").

  • The "Paired Samples Effect Sizes" table gives standardized measures of effect size, we will use the "Cohen's d" measure because it is the simplest and most common. The effect size is shown in the "Point estimate" column.

  • There are two distinct ways to think about effect size for a within-subject IV:

    A. We might argue that the effect size should be comparable across different studies, thus it should not be dependent on the study design (i.e., between vs within-subjects).

    • in this case we should ignore the dependency between cases and pool the variance across the two conditions when computing the effect size
    • you can get this measure by selecting "Average of variances" in the effect size options for the test (instead of "Standard deviation of the difference")

    B. Alternatively, we might argue that the effect size should factor in the dependency between scores in different conditions
    - In this case we should factor in the correlation between the scores using the "Standard deviation of the difference" option for effect size (this is what the instructions said to select). This effect size is sometimes referred to as dz (Lakens, 2013).

  • Which effect size is correct?

    • It depends on your research question and, to some extent, expectations in the specific area of study.
    • Some research questions might be specific to repeated measures designs, and in that case #2 makes sense.
    • But #1 makes sense when you intend the effect size to be comparable across different study designs.
    • The important thing is to be clear about how effect size is calculated when you report it. E.g. Methods: "Effect size for paired comparisons is reported as d = (mean difference)/(standard deviation of the difference)(Lakens, 2013)"
    • For this activity (and your research reports) report dz whenever you compare levels of a within-subjects variable.

General guide - how to report the result of a t-test comparing two means in a within-subjects variable.

A report of the result of a comparison like this should include:
1. Type of comparison and effect size measure (this can be in the Methods section. e.g. "Means were compared by an paired samples t-test (two-sided). Effect size for paired comparisons is reported as d = (mean difference)/(standard deviation of the difference) as described by Lakens (2013).")
2. In the Results section: Means and SDs for each condition, and for the difference between conditions.
3. In the Results section: Test statistic with degrees of freedom and p-value, specify one/two-sided.
4. In the Results section: Effect size.
